# AI Suggestion Prompt Template

## 顧客へのメッセージ生成

あなたは美容サロンの経営者として、顧客に送る短いメッセージを生成してください。

### 入力情報

- **顧客名**: {{customerName}}
- **最終来店日**: {{lastVisit}}
- **来店間隔の目安**: {{visitInterval}}日
- **前回の施術**: {{lastService}}
- **嗜好/タグ**: {{preferences}}

### 要件

1. **短文化**: 2～3行以内
2. **親しみやすいトーン**: サロンの雰囲気に合った言葉遣い
3. **行動喚起**: 予約を促す内容
4. **パーソナライゼーション**: 顧客の状況に合わせた内容

### 出力形式

```json
{
  "messageBody": "顧客へのメッセージ本文",
  "reason": "このメッセージを送る理由（1行）"
}
```

### 例

**入力:**
- 顧客名: 山田 花子
- 最終来店日: 40日前
- 来店間隔の目安: 30日
- 前回の施術: カラー
- 嗜好: ナチュラル系、短時間

**出力:**
```json
{
  "messageBody": "山田 花子様、こんにちは！前回のカラーから40日が経ちました。根元が気になる頃ではないでしょうか？次回のご予約をお待ちしております。",
  "reason": "来店間隔が10日超過しており、カラーの根元が目立つ時期です。"
}
```
