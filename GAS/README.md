# 脱毛サロン業績管理テンプレート

## 📊 概要

このテンプレートは、脱毛サロンの業績管理を効率化するためのGoogle Sheetsアドオンです。

## 🚀 インストール手順

1. Google Sheetsで新規スプレッドシートを開く
2. **拡張機能 → Apps Script** を選択
3. 以下のファイルを作成してコードを貼り付け：
   - `appsscript.json`（プロジェクト設定から「マニフェストエディタで表示」をON）
   - `main.gs`
   - `constants.gs`
   - `utils_formatting.gs`
   - `utils_validation.gs`
   - `utils_sheets.gs`
   - `builders_dashboard.gs`
   - `builders_sales.gs`
   - `builders_ads.gs`
   - `builders_customers.gs`
   - `builders_expenses.gs`
   - `builders_profit.gs`
   - `builders_staff.gs`
   - `builders_actions.gs`
   - `seed_sampleData.gs`
4. **保存**（Ctrl+S / Cmd+S）
5. スプレッドシートに戻り、**再読み込み**（F5 / Cmd+R）
6. 上部メニューに **「📊 業績テンプレ」** が表示される

## 🎯 使い方

### 初回セットアップ
1. **📊 業績テンプレ → 🏗️ 初期構築（全シート生成）** を実行
2. **📊 業績テンプレ → 📝 ダミーデータ投入** を実行（動作確認用）
3. ダミーデータを削除して、実際のデータを入力開始

### 日常の使い方
- **売上シート**：日々の売上を入力
- **広告シート**：広告媒体ごとの実績を月単位で入力
- **顧客シート**：顧客情報と継続状況を管理
- **経費シート**：月次の固定費・変動費を入力
- **スタッフシート**：スタッフごとの実績を管理
- **ダッシュボード**：自動で主要指標を集計

## 📈 追いかけるべき重要指標（KPI）

### 🎯 最優先指標（週次確認）

1. **営業利益**
   - 目標：月30万円以上
   - 計算：売上 - 経費
   - 見方：赤字の場合は即座に経費削減または売上向上策を実施

2. **利益率**
   - 目標：20%以上
   - 計算：営業利益 ÷ 売上
   - 見方：15%以下なら経費構造を見直し

3. **次回予約率**
   - 目標：80%以上
   - 計算：次回予約あり顧客 ÷ 全顧客
   - 見方：低下したら施術品質や接客を改善

### 📊 重要指標（月次確認）

4. **継続率**
   - 目標：85%以上
   - 計算：継続顧客 ÷ 全顧客
   - 見方：70%以下は解約理由を分析

5. **CPA（顧客獲得単価）**
   - 目標：15,000円以下
   - 計算：広告費 ÷ 成約数
   - 見方：効率の悪い媒体は予算を削減

6. **ROI（投資対効果）**
   - 目標：150%以上
   - 計算：(LTV - CPA) ÷ CPA
   - 見方：負の値は即座に広告を見直し

### 💰 収益性指標（月次確認）

7. **新規来店数**
   - 目標：月20名以上
   - 見方：減少傾向なら広告強化

8. **平均客単価**
   - 目標：150,000円以上
   - 見方：低下したら高額コースの提案を強化

9. **スタッフ別売上**
   - 目標：一人あたり月100万円以上
   - 見方：低い場合は研修や配置転換を検討

## 📋 月次レビューのやり方

### 毎月5日：前月実績の確認

1. **ダッシュボード**を開く
2. 以下を確認：
   - ✅ 営業利益が目標達成か？
   - ✅ 利益率は20%以上か？
   - ✅ 継続率は85%以上か？
   - ✅ 広告のROIは黒字か？

3. **改善アクションシート**に記録：
   - 課題：「継続率が75%に低下」
   - 改善施策：「施術後のフォローアップ強化」
   - 担当者：「店長」
   - 実施日：「2025/02/10」

4. 翌月に効果を測定して記録

## 💡 よくある改善パターン

### 利益率が低い場合
- ❌ 人件費が売上の40%超 → 採用見直し
- ❌ 広告費が売上の30%超 → 媒体を絞る
- ❌ 材料費が高い → 仕入れ先の見直し

### 継続率が低い場合
- 施術技術の向上（研修実施）
- 次回予約のタイミング改善
- カウンセリングの質向上
- 価格設定の見直し

### 新規獲得が不足
- Instagram広告の強化
- ホットペッパーの掲載順位UP
- 紹介キャンペーン実施
- Googleマイビジネスの最適化

## 🔧 トラブルシューティング

### エラーが出る場合
- **#REF!**：参照先シートが存在しない → 初期構築を再実行
- **#DIV/0!**：ゼロ除算 → データを入力してから確認
- **#VALUE!**：型エラー → 数値項目に文字が入っていないか確認

### データをリセットしたい
1. 全シートを削除
2. **📊 業績テンプレ → 🏗️ 初期構築（全シート生成）**
3. **📊 業績テンプレ → 📝 ダミーデータ投入**（動作確認用）
